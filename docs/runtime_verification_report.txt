5. Known Limitations

- Complex PPT files with heavy template styling, image-based text, or unusual layout metadata can still produce garbled extracted text
- For the final presentation, a text-heavy .pptx with simpler structure is recommended
- During long-running dev sessions, Next.js dev hot state can occasionally become stale and require a manual dev server restart
