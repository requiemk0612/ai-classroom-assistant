Codex Build Prompts For AI Classroom Assistant Demo

Usage rule
- These prompts are for an AI coding assistant.
- Follow the prompts in the exact order.
- All code comments and variable names must use simple English.
- All user-facing text must be Chinese.
- Both teacher side and student side must be web pages.
- All documents must stay in .txt format.
- Keep the implementation lightweight and demo-first.
- Follow docs/demo_technical_plan.txt exactly.
